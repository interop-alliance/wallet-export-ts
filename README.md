## Install

- Node.js 14+ is recommended.

### NPM

To install via NPM:

```
npm install @interop/isomorphic-lib-template
```

### Development

To install locally (for development):

```
git clone https://github.com/interop-alliance/isomorphic-lib-template.git
cd isomorphic-lib-template
npm install
```
