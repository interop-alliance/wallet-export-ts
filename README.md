## Install

- Node.js 14+ is recommended.

### NPM

To install via NPM:

```
npm install @interop/wallet-export-ts
```

### Development

To install locally (for development):

```
git clone https://github.com/interop-alliance/wallet-export-ts.git
cd wallet-export-ts
npm install
```
